N

Terms Defination
native A Java keyword used to declare a method whose implementation is written in another language, usually C or C++.
new A Java keyword used to create a new object or array instance.
Namespace A logical grouping that helps organize identifiers and avoid naming conflicts; in Java, packages serve this purpose.
Naming Convention Standard rules for naming classes, methods, variables, and packages to improve readability and consistency.
Native Method A method declared with the native keyword and implemented outside Java.
Native Library An external library (such as a DLL or shared object file) containing native code used by Java applications.
Narrowing Conversion Conversion of a larger data type to a smaller one, such as double to int, which may lose data.
Nested Class A class declared inside another class.
Nested Interface An interface declared within another class or interface.
Network Programming Writing Java programs that communicate over a network using sockets, URLs, and related APIs.
NIO (New I/O) Java's enhanced input/output API that provides buffers, channels, and non-blocking operations.
Non-Access Modifier Modifiers like static, final, abstract, and native that affect behavior rather than access level.
Non-Blocking I/O I/O operations that do not force a thread to wait until the operation completes.
Null A special literal representing the absence of an object reference.
Null Reference A reference variable that points to no object (null).
Null Check Verification that a reference is not null before using it.
NullPointerException Runtime exception thrown when attempting to use a null reference as if it referred to an object.
Numeric Promotion Automatic conversion of smaller numeric types to larger types during arithmetic operations.
Number Abstract superclass of numeric wrapper classes such as Integer, Double, and Float.
NumberFormatException Exception thrown when converting a string to a number fails due to invalid format.
NumberFormat Class used for formatting and parsing numeric values.
NavigableMap Interface extending Map that provides navigation methods for sorted key-value mappings.
NavigableSet Interface extending SortedSet that provides navigation methods for sorted collections.
Node A basic element in a data structure such as a linked list, tree, or graph.
NetworkInterface Class representing a network interface card (NIC) or network adapter.
NegativeArraySizeException Exception thrown when an array is created with a negative size.
NoSuchElementException Exception thrown when attempting to access an element that does not exist.
NoSuchFieldException Exception thrown when a requested field cannot be found through reflection.
NoSuchMethodException Exception thrown when a requested method cannot be found through reflection.
NoSuchAlgorithmException Exception thrown when a cryptographic algorithm is unavailable.
NoSuchFileException Exception thrown when a specified file cannot be found.
NoClassDefFoundError Error indicating that a class needed at runtime cannot be located.
NamingException Exception used in Java Naming and Directory Interface (JNDI) operations.
NotSerializableException Exception thrown when an object that does not implement Serializable is serialized.
NotBoundException Exception thrown when an RMI registry lookup fails because a name is not bound.
Notify Method (notify()) that wakes up one thread waiting on an object's monitor.
NotifyAll Method (notifyAll()) that wakes up all threads waiting on an object's monitor.
Native Thread An operating-system thread used by the JVM to execute Java threads.
NewFixedThreadPool ExecutorService factory method that creates a thread pool with a fixed number of threads.
NewCachedThreadPool ExecutorService factory method that creates a thread pool that can grow as needed.
NewSingleThreadExecutor ExecutorService factory method that creates a single-worker thread executor.
NewWorkStealingPool ExecutorService factory method that creates a work-stealing thread pool for parallel tasks.
Native Memory Memory allocated outside the Java heap, often used by native code.
Non-Heap Memory JVM memory area used for class metadata, code cache, and other runtime data.
Null Object Pattern Design pattern that uses a special object instead of null to avoid null checks.
Named Annotation often used in dependency injection frameworks to identify a specific bean or component.
Nonnull Annotation indicating that a variable, parameter, or return value should never be null.
Nullable Annotation indicating that a variable, parameter, or return value may be null.
Normalizer A utility class in java.text used to normalize Unicode text into a standard form.
NamespaceContext An interface in XML processing that provides mappings between namespace prefixes and namespace URIs.
NodeList A collection of DOM nodes returned by XML document operations.
NodeIterator An interface used to iterate through nodes in a DOM tree.
NamedNodeMap A DOM interface representing a collection of nodes that can be accessed by name.
Notation A DOM interface representing a notation declared in a document type definition (DTD).
NamingEnumeration An interface used by JNDI to iterate through naming and directory service results.
Name An interface in JNDI representing a generic name composed of multiple components.
NameParser An interface that parses string names into JNDI Name objects.
NameClassPair A JNDI class containing a name and its associated class name.
NamingManager A JNDI utility class that manages naming and directory operations.
NamingSecurityException Base exception for security-related JNDI errors.
NamingListener An event listener interface for receiving JNDI naming events.
NamespaceSupport A helper class for managing XML namespaces during SAX parsing.
NClob An interface representing a national character large object in JDBC databases.
NString A JDBC concept referring to Unicode string data stored in a database.
NetworkChannel An interface for channels connected to network sockets.
NotDirectoryException Exception thrown when a path expected to be a directory is not one.
NotLinkException Exception thrown when a path expected to be a symbolic link is not one.
NonReadableChannelException Runtime exception thrown when attempting to read from a non-readable channel.
NonWritableChannelException Runtime exception thrown when attempting to write to a non-writable channel.
NotYetBoundException Exception thrown when an operation requires a bound channel that has not yet been bound.
NotYetConnectedException Exception thrown when an operation requires a connected channel that is not yet connected.
NioSocketImpl Internal JVM socket implementation used by Java's networking APIs.
Notification A JMX class representing a management event emitted by an MBean.
NotificationListener Interface that receives JMX notifications.
NotificationFilter Interface used to filter JMX notifications before delivery.
NotificationBroadcaster JMX interface for objects capable of sending notifications.
NotificationEmitter Extended JMX notification broadcaster interface.
NotificationResult JMX class containing a set of notifications returned from a remote connector.
NotificationChain Internal mechanism used in event propagation and monitoring systems.
NoSuchAttributeException JMX exception thrown when a requested attribute does not exist.
NoSuchObjectException RMI exception indicating that a remote object no longer exists.
NoRouteToHostException Networking exception indicating that a destination host cannot be reached.
NestingKind Enum describing the nesting type of a Java class (top-level, member, local, anonymous).
NullType Type model interface representing the type of the null literal.
NameService Internal networking service used for host-name resolution.
NativeBuffer Internal JVM buffer allocated in native memory.
NativeMemoryTracking (NMT) JVM feature used to monitor native memory usage.
NewLine Line separator character sequence (\n, \r\n) used in text processing.
NonCapturingGroup A regular-expression group that groups patterns without capturing matched text.
NamedCapturingGroup A regular-expression group identified by a name instead of a number.
Nashorn JavaScript engine included in older Java versions (removed after JDK 14).
NashornScriptEngine Script engine implementation used to execute JavaScript through Nashorn.
NashornException Base exception class used by the Nashorn engine.
NIO Buffer A memory container used by NIO channels for efficient data transfer.
NIO Channel A connection to a file, socket, or device that supports data transfer operations.
NIO Selector Component that enables a single thread to monitor multiple channels.
