A

Terms Defination
Abstract An abstract class or method is incomplete and cannot be used directly. It helps achieve abstraction in Java.
Abstract Method An abstract method has no body and must be implemented by subclasses.
Abstract Class An abstract class cannot be instantiated and may contain both abstract and concrete methods.
Abstract Data Type (ADT) An ADT defines data and operations without specifying implementation details.
Abstract Factory Pattern A design pattern used to create related objects without specifying their exact classes.
Abstract Collection An abstract class that provides basic functionality for collection classes.
Abstract List An abstract implementation of the List interface used as a base for custom lists.
Abstract Map An abstract implementation of the Map interface that simplifies custom map creation.
Abstract Queue An abstract class that provides common queue operations and behavior.
Abstract Set An abstract implementation of the Set interface.
AbstractExecutorService An abstract class that provides default implementations for ExecutorService methods.
AbstractOwnableSynchronizer A synchronization utility class used in concurrent programming.
AbstractQueuedSynchronizer A framework for building locks and synchronization tools.
AbstractQueuedLongSynchronizer A version of AQS that uses long values for synchronization state.
AbstractSelector An abstract class used in Java NIO for channel selection.
AbstractStringBuilder The parent class of StringBuilder and StringBuffer.
AbstractTableModel A base class used to create custom table models in Swing.
Access Modifier A keyword that controls the visibility of classes, methods, and variables.
Accessibility Accessibility determines where a class member can be accessed.
AccessibleObject A superclass for reflection objects that allows runtime access to class members.
Accessor Method A method used to retrieve the value of a private variable.
Action An operation performed by a user or program that may trigger an event.
ActionEvent An event generated when a user performs an action such as clicking a button.
ActionListener An interface used to receive and handle ActionEvents.
Activation The process of making an object available for remote access.
Activatable A class used in RMI activation services.
Activator A component responsible for activating objects in certain frameworks.
Adapter A class that allows incompatible interfaces to work together.
Adapter Class A class providing default implementations of listener methods.
Adapter Pattern A structural design pattern that converts one interface into another.
Aggregation A "has-a" relationship where objects can exist independently.
Algorithm A step-by-step procedure used to solve a problem.
Alias An alternative reference to the same object in memory.
Allocation The process of reserving memory for variables and objects.
AlphaComposite A class used for advanced image compositing in Java graphics.
Ambiguity A situation where the compiler cannot determine which method or constructor to use.
Annotation Metadata added to Java code using the @ symbol.
Annotation Processing The process of examining and handling annotations during compilation.
Annotation Type A special interface used to define custom annotations.
AnnotationFormatError An error indicating a problem with annotation data.
Anonymous Class A class without a name created and instantiated in a single statement.
Anonymous Inner Class An unnamed inner class used for one-time implementations.
Ant A Java-based build automation tool.
API A collection of predefined classes, interfaces, and methods.
Applet A small Java program designed to run inside a browser.
AppletContext An interface that enables communication between an applet and its environment.
AppletStub An interface that provides services required by an applet.
AppletViewer A tool used to run and test applets without a browser.
AppContext A context object used internally by AWT applications.
Application A standalone Java program that starts execution from the main() method.
Append The process of adding data to the end of existing content.
Appendable An interface for objects that support appending characters.
ArithmeticException An exception caused by invalid arithmetic operations such as division by zero.
Arithmetic Operator Operators used to perform mathematical calculations.
Array A fixed-size collection of elements of the same data type.
Array Bounds The valid index range of an array.
Array Copy The process of copying elements from one array to another.
Array Index The position number used to access an array element.
Array Length The total number of elements an array can store.
ArrayBlockingQueue A thread-safe queue with a fixed capacity.
ArrayDeque A resizable array implementation of the Deque interface.
ArrayList A dynamic array implementation of the List interface.
Arrays A utility class that provides methods for array manipulation.
Arrays.asList() A method that converts an array into a fixed-size List.
ArrayStoreException An exception thrown when an incompatible object is stored in an array.
ASCII A character encoding standard that assigns numeric values to characters.
Assertion A debugging statement used to verify assumptions in code.
AssertionError An error thrown when an assertion condition evaluates to false.
Assignment The process of storing a value into a variable.
Assignment Operator An operator used to assign values to variables.
Associativity The rule that determines the order of operator evaluation.
Asynchronous Programming A technique where tasks execute independently without blocking.
Aspect A module in AOP that contains cross-cutting functionality.
AspectJ A popular framework for Aspect-Oriented Programming in Java.
Aspect-Oriented Programming (AOP) A paradigm that separates cross-cutting concerns from business logic.
Atomic Variable A thread-safe variable supporting atomic operations.
AtomicBoolean A thread-safe wrapper class for boolean values.
AtomicInteger A thread-safe wrapper class for integer values.
AtomicLong A thread-safe wrapper class for long values.
AtomicReference A thread-safe wrapper class for object references.
Attribute A variable declared within a class that stores object data.
Auto-boxing Automatic conversion of primitive types into wrapper objects.
AutoCloseable An interface whose objects are automatically closed after use.
Auto Increment The process of increasing a variable's value by one using ++.
Auto Decrement The process of decreasing a variable's value by one using --.
Autowired A Spring annotation used for automatic dependency injection.
AWT Abstract Window Toolkit used for GUI development.
AWT Component A visual GUI element such as a button or text field.
AWT Event An event generated by user interaction in an AWT application.
AWT Graphics A class used for drawing shapes, text, and images.
AWT Layout A manager responsible for arranging GUI components.
AWT Toolkit A utility class providing access to platform-specific resources.
AWTEvent The superclass of all AWT events.
AWTException An exception thrown when an AWT operation fails.
Await A method that pauses a thread until a condition becomes true.
AwaitTermination A method that waits for submitted tasks to complete before shutdown.
