G

Terms Defination
Garbage Collection The automatic process of removing unused objects from memory to free resources.
Garbage Collector The JVM component responsible for performing garbage collection.
Garbage Collection Cycle A complete execution of the garbage collection process.
Garbage First (G1) Collector A modern JVM garbage collector designed for low-pause-time applications.
Gateway A component that acts as an entry point between different systems or networks.
GC Root An object used as a starting point during garbage collection.
General Exception A broad exception type used for error handling.
Generalization An OOP concept where common features are extracted into a superclass.
Generated Class A class automatically created by a compiler, framework, or tool.
Generation (GC) A memory region used by generational garbage collectors.
Generational GC A garbage collection technique that divides memory into generations.
Generic Array An array that uses generic type parameters.
Generic Class A class that can work with different data types using type parameters.
Generic Collection A collection that uses generics to ensure type safety.
Generic Interface An interface that uses generic type parameters.
Generic Method A method that introduces its own type parameters.
Generic Type A parameterized type used with generics.
Generic Wildcard A special symbol (?) used in generic type declarations.
Generics A feature that enables type safety by allowing parameterized types.
get() Method A method commonly used to retrieve the value of an object or property.
getClass() A method that returns the runtime class of an object.
getName() A method used to retrieve the name of an object, class, or thread.
getProperty() A method used to retrieve system properties.
Getter Method A method used to access the value of a private variable.
Global Variable A variable accessible from multiple parts of a program (not directly supported in Java).
Glyph Vector A collection of glyphs used for advanced text rendering.
GraalVM A high-performance runtime that supports Java and other programming languages.
Graph A data structure consisting of vertices and edges.
Graph Traversal The process of visiting all nodes in a graph.
Graphical Object Any object that can be drawn on a graphical interface.
Graphics A class used for drawing shapes, text, and images in GUI applications.
Graphics2D An advanced version of Graphics that provides enhanced drawing capabilities.
Graphics Configuration A class describing graphics device capabilities.
Graphics Device A device such as a monitor or printer used for rendering graphics.
Graphics Environment A collection of graphics devices available on the system.
Gray Code A binary numbering system where consecutive values differ by only one bit.
Grid Cell An individual position within a grid layout.
Grid Coordinate A row and column position in a grid-based layout.
GridBagConstraints A class used to specify constraints in GridBagLayout.
GridBagLayout A flexible layout manager that arranges components in a grid.
GridLayout A layout manager that arranges components in rows and columns.
GregorianCalendar A calendar implementation based on the Gregorian calendar system.
Group A collection of related objects or components.
GroupLayout A layout manager commonly used in GUI builders such as NetBeans.
Grouping Collector A Stream API collector used to group elements based on a key.
Growth Factor The rate at which a dynamic data structure increases in size.
Guarded Block A block of code that waits until a specific condition becomes true.
GUI (Graphical User Interface) A user interface that allows interaction through graphical components.
GUI Component A visual element such as a button, label, or text field.
GUI Container A component capable of holding other GUI components.
GUI Event An event generated by user interaction with a GUI component.
GUI Framework A library used to build graphical user interfaces.
GUI Thread The thread responsible for handling graphical user interface events.
GZIPInputStream A stream used to read compressed GZIP data.
GZIPOutputStream A stream used to write compressed GZIP data.
Game Loop A loop that continuously updates and renders a game.
Geometric Shape A graphical object such as a rectangle, circle, or polygon.
Graphics Context An object containing information needed for drawing operations.
Graphics Primitive A basic graphical element such as a line, rectangle, or circle.
Graphics Rendering The process of displaying graphical content on the screen.
Grid-Based Layout A layout structure that organizes components in rows and columns.
Group Thread A collection of threads managed as a single unit.
GroupingBy() A Stream API collector method used to group elements.
Generic Programming A programming technique that allows code to work with multiple data types.
Garbage-Free Programming A coding approach that minimizes object creation to reduce garbage collection overhead.
goto A reserved keyword in Java that is not used and has no functionality.
