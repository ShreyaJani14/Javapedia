O

Terms Defination
Object The root class of the Java class hierarchy. Every Java class directly or indirectly inherits from Object.
Object Class The java.lang.Object class that provides methods such as equals(), hashCode(), and toString().
Object-Oriented Programming (OOP) A programming paradigm based on objects, encapsulation, inheritance, abstraction, and polymorphism.
Object Reference A variable that stores the memory address/reference of an object.
Object Creation The process of creating an object using the new keyword.
Object Serialization Converting an object into a byte stream for storage or transmission.
Object Deserialization Reconstructing an object from a serialized byte stream.
ObjectInput Interface for reading primitive data types and objects from a stream.
ObjectOutput Interface for writing primitive data types and objects to a stream.
ObjectInputStream Class used to deserialize objects from an input stream.
ObjectOutputStream Class used to serialize objects to an output stream.
ObjectStreamClass Serialization descriptor containing metadata about a serialized class.
ObjectStreamField Describes a serializable field within a class.
Observer A design pattern where objects are notified automatically when another object's state changes.
Observable Legacy Java class supporting the Observer pattern.
Optional Container object that may or may not contain a non-null value.
OptionalDouble Specialized Optional container for double values.
OptionalInt Specialized Optional container for int values.
OptionalLong Specialized Optional container for long values.
Overloading Defining multiple methods with the same name but different parameter lists.
Overriding Providing a new implementation of an inherited method in a subclass.
Operator A symbol that performs operations on operands.
Operand A value on which an operator acts.
OutputStream Abstract class representing a destination for byte output.
OutputStreamWriter Bridge between character streams and byte streams.
OutputMismatchException Exception indicating output data does not match expected values.
OpenOption Interface defining options for opening files in NIO.
OpenDataException Exception used in JMX Open MBean operations.
OpenMBeanAttributeInfo JMX interface describing an Open MBean attribute.
OpenMBeanConstructorInfo JMX interface describing an Open MBean constructor.
OpenMBeanInfo JMX interface containing Open MBean metadata.
OpenMBeanOperationInfo JMX interface describing an Open MBean operation.
OpenMBeanParameterInfo JMX interface describing an Open MBean parameter.
OpenType Base class representing the type of Open MBean values.
OperatingSystemMXBean JMX interface providing operating-system information.
OffsetDateTime Date-time class representing a date and time with an offset from UTC.
OffsetTime Time class representing a time with an offset from UTC.
Ordinal Position index of an enum constant returned by ordinal().
Outer Class A class that contains one or more nested classes.
OutOfMemoryError Error thrown when the JVM cannot allocate enough memory.
Override Annotation (@Override) Annotation indicating that a method overrides a superclass method.
ObjectFactory Factory class responsible for creating objects dynamically.
ObjectName JMX class identifying a managed bean uniquely.
ObjectInstance JMX class representing an MBean instance.
ObjectFactoryBuilder Interface used to create object factories.
OID (Object Identifier) Unique identifier used in security and directory services.
Opaque Value A value whose internal representation is hidden from the user.
Octal Literal Integer literal represented in base-8 notation.
Order Comparator Mechanism used to determine the ordering of objects.
Ordered Collection Collection that maintains element order.
Overflow Condition when a value exceeds the storage capacity of its data type.
One-to-One Mapping Relationship where one object is associated with exactly one other object.
One-to-Many Mapping Relationship where one object is associated with multiple objects.
Object Pool Design pattern that reuses objects instead of creating new ones repeatedly.
Open Connection An active network or database connection.
Open File A file currently accessed through a stream or channel.
OpenJDK The open-source reference implementation of the Java Platform.
ORM (Object-Relational Mapping) Technique that maps Java objects to relational database tables.
OXM (Object XML Mapping) Technique that maps Java objects to XML structures.
ObjectStreamException Base exception class for exceptions related to object streams and serialization.
OptionalDataException Exception thrown during deserialization when primitive data is found instead of an expected object or the end of stream is reached.
ObjectStreamException Parent class for serialization-related exceptions.
ObjectStreamClass.lookup() Method used to obtain serialization metadata for a class.
ObjectStreamField Represents a serializable field of a class during serialization.
ObjectInputValidation Interface that allows an object to validate itself after deserialization.
ObjectInputFilter Interface used to filter classes and objects during deserialization for security.
ObjectInputFilter.Status Enum representing whether a deserialized object is allowed or rejected.
ObjectInputFilter.Config Utility class used to configure object input filters.
ObjectFactory Interface/class used to create objects dynamically, commonly used in JNDI and XML processing.
ObjectFactoryBuilder Interface used to build object factories.
ObjectInstance JMX class representing the name and class of an MBean.
ObjectName JMX class used to uniquely identify an MBean.
ObjectNamePattern An ObjectName containing wildcards used to match multiple MBeans.
ObjectVariableTree Compiler tree API representation of a variable declared in source code.
ObjectType Represents the type information of an object in reflection or compiler APIs.
ObjectStreamConstants Interface containing constants used by Java serialization streams.
ObjectStreamClassException Exception related to invalid serialization class information.
Object Cloning Creating an exact copy of an existing object using clone().
Cloneable Marker interface that allows an object to be cloned.
CloneNotSupportedException Exception thrown when cloning is attempted on an object that does not support it.
Object Equality Comparison of two objects using equals() method.
Object Identity Comparison of whether two references point to the same object using ==.
Object Lock Lock associated with an object used for synchronization.
Object Monitor JVM mechanism that controls synchronized access to an object.
Object Header JVM memory area storing metadata about an object.
Object Heap Memory area where Java objects are stored.
Object Allocation Process of reserving memory for a new object.
Object Lifetime Time period from object creation until garbage collection removes it.
Object Graph Network of objects connected through references.
Object Pool Pattern Design pattern that reuses a set of initialized objects.
Object Adapter Pattern Design pattern that converts one object's interface into another interface.
Object Factory Pattern Design pattern that creates objects without exposing creation logic.
Object Relational Mapping (ORM) Technique for mapping Java objects to database tables.
Object XML Mapping (OXM) Technique for mapping Java objects to XML documents.
Object-Relational Impedance Mismatch Difference between object-oriented Java models and relational databases.
Observer Pattern Behavioral design pattern where observers receive updates when a subject changes.
Observable Class Legacy Java class that supports the Observer pattern.
Observer Interface Interface defining an object that receives updates from an Observable.
Overflow Situation where a numeric value exceeds the maximum range of a data type.
Underflow Situation where a numeric value becomes too small to represent accurately.
Octal Number System Number system using base 8, represented in Java with a leading 0.
Octal Literal Integer literal written using octal notation (example: 012).
Operator Precedence Rules deciding the order in which operators are evaluated.
Operator Associativity Rule determining evaluation direction when operators have equal precedence.
OR Operator (||) Logical OR operator.
OR Assignment (|=) Bitwise OR assignment operator.
Overloaded Method Method with the same name but different parameters.
Overloaded Constructor Constructor with different parameter lists in the same class.
Overload Resolution Process of choosing the correct overloaded method during compilation.
Override Replacing inherited method implementation in a subclass.
Overriding Method Method in a child class that provides a new implementation of a parent method.
Override Annotation @Override annotation that tells the compiler a method overrides another method.
Overriding Rules Rules such as same signature, cannot reduce visibility, and cannot override final methods.
OpenJDK Open-source implementation of Java Development Kit.
Open Source Java Java implementations whose source code is publicly available.
Open File Descriptor Operating system resource representing an opened file.
Open Connection Active connection to a resource such as a database or network service.
Open Channel Active NIO channel used for data transfer.
OpenOption Interface defining file opening options in NIO.
OpenMode Mode describing how a resource is opened (read, write, append, etc.).
OutputStream Abstract class for writing byte data.
OutputStreamWriter Converts character output into byte output using a charset.
Output Buffer Temporary memory area used to store output before writing.
Output Encoding Process of converting characters into bytes for storage/transmission.
OutputMismatchException Exception indicating that generated output does not match expected output.
OffsetDateTime Date and time class with a UTC offset.
OffsetTime Time class with a UTC offset but without a date.
OffsetClock Clock that applies an offset to another clock.
Ordinal Value Integer position of an enum constant starting from zero.
Ordered Collection Collection that maintains insertion or sorted order.
Ordered Stream Stream whose elements have a defined encounter order.
OperatingSystemMXBean Interface that provides JVM access to operating system information.
OSGi (Open Service Gateway Initiative) Modular Java framework for developing dynamic applications.
