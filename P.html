Terms Defination
package Java keyword used to group related classes and interfaces.
private Access modifier that allows access only within the same class.
protected Access modifier allowing access within the same package and subclasses.
public Access modifier allowing access from anywhere.
polymorphism OOP concept where one interface can have many implementations.
parameter Variable declared in a method definition that receives a value.
parameterized constructor Constructor that accepts arguments.
pass by value Java method argument passing mechanism where copies of values are passed.
parse Converting data from one format into another, such as String to number.
Parser Component that analyzes input data according to grammar rules.
Parsing Process of reading and interpreting data.
parent class The superclass from which a child class inherits.
parentheses Symbols () used for method calls, grouping expressions, and conditions.
primitive data type Basic built-in data types like int, char, boolean, and double.
primitive wrapper class Class representation of primitive values (Integer, Double, etc.).
PrintStream Class used to print formatted output.
PrintWriter Class used to write formatted character output.
printf() Method used to print formatted output.
println() Method that prints output followed by a newline.
print() Method that prints output without adding a newline.
Process Represents an operating system process created by Java.
ProcessBuilder Class used to create and control external processes.
ProcessHandle Interface used to manage and inspect operating system processes.
ProcessHandle.Info Interface containing process information.
ProcessEnvironment Internal class representing environment variables.
Proxy Class that creates dynamic proxy objects.
Proxy pattern Design pattern that provides a substitute object controlling access to another object.
Predicate Functional interface that takes input and returns true/false.
Predicate chaining Combining multiple predicates using logical operations.
Producer-Consumer Pattern Concurrency pattern where producers create data and consumers process it.
Producer thread Thread that generates data for another thread.
Program counter (PC Register) JVM memory area storing address of current instruction.
Package-private Default access level when no modifier is specified.
Package declaration Statement that declares the package of a class.
Package import Importing classes from another package.
Package-info.java Special file used for package documentation and annotations.
Path Interface representing file system paths in NIO.
PathMatcher Interface used to match paths against patterns.
Paths Utility class for creating Path objects.
PathVariable Framework term for variable extracted from a URL path.
Pattern Class representing regular expressions.
Pattern Matching Technique of checking data against a pattern.
PatternSyntaxException Exception thrown for invalid regular expressions.
Polymorphic method Method that behaves differently depending on object type.
Port Network endpoint used for communication.
Portability Ability of Java programs to run on different platforms.
Platform Operating environment where Java runs.
Platform-independent Java feature allowing programs to run on any OS with JVM.
Platform Thread Traditional JVM-managed thread.
Project Loom Java project introducing lightweight virtual threads.
ProxySelector Class that selects network proxies.
Protocol Rules defining communication between systems.
Provider Implementation provider for Java security, cryptography, or services.
Provider class Security provider implementation class.
Properties Class storing configuration data as key-value pairs.
Property Named value associated with an object or configuration.
PropertyChangeListener Listener interface for property change events.
PropertyChangeEvent Event object describing a property change.
PropertyDescriptor Class describing JavaBean properties.
Pseudo-random number Number generated using deterministic algorithms.
Pseudo-code Informal description of program logic.
PipedInputStream Input stream connected to a piped output stream.
PipedOutputStream Output stream connected to a piped input stream.
PipedReader Character input stream connected to a writer.
PipedWriter Character output stream connected to a reader.
PriorityQueue Queue that orders elements by priority.
PriorityBlockingQueue Thread-safe priority queue for concurrent programming.
Priority Importance level used in scheduling tasks or threads.
Priority of Thread Numeric value indicating thread scheduling preference.
PackageManager Android/Java-related component managing packages.
Persistence Saving data so it remains after program termination.
Persistent Object Object whose state is stored permanently.
PreparedStatement JDBC interface for executing precompiled SQL queries.
Procedure Stored database operation executed through JDBC.
Programming Language Formal language used to write software instructions.
Programming Interface Set of rules for interaction between software components.
Plugin Extension that adds functionality to an application.
Parser Generator Tool that generates parsers from grammar definitions.
Package Scanner Tool that searches classes inside packages.
Permission Security rule controlling access to resources.
Permissions Collection of security permissions.
Policy File Security configuration file defining permissions.
Protection Domain Security boundary containing permissions for code.
Proxy Authentication Authentication required for proxy servers.
ProxyConnection Connection through a proxy server.
Phantom Reference Reference type used for advanced garbage collection tracking.
PhantomReachable State related to phantom references.
Pointer Memory reference concept; Java does not expose raw pointers.
Pointer Arithmetic Changing memory addresses; not supported in Java.
Package Hierarchy Structure of packages and subpackages.
Parameter Passing Sending values/objects to methods.
Passivation Saving object state temporarily, often in distributed systems.
Post-increment (++) Operator that increases value after use.
Pre-increment (++) Operator that increases value before use.
Post-decrement (--) Operator that decreases value after use.
Pre-decrement (--) Operator that decreases value before use.
Prefix notation Operator placed before operand.
Postfix notation Operator placed after operand.
Primitive Casting Converting one primitive type to another.
Protected Constructor Constructor accessible by subclasses and package classes.
Package Locking JVM mechanism related to package/class loading.
Package A namespace used to organize related classes and interfaces.
Package Annotation Annotation applied to an entire package using package-info.java.
Package Member A class or interface declared inside a package.
Package Object A concept used in some languages; Java does not support package objects.
Package Sealing Restricting classes in a package to come from the same code source.
Package Versioning Managing different versions of Java packages.
Package Dependency A relationship where one package requires another package.
Package Fragment Term mainly used in modular systems for partial package definitions.
Parameter List List of variables passed to a method or constructor.
Parameter Type The data type of a method parameter.
Parameter Passing Mechanism The way values are transferred to methods in Java (always pass-by-value).
Parameter Object Pattern Design pattern that groups multiple parameters into one object.
Parent Interface Interface inherited by another interface.
Parent Reference Reference variable pointing to a parent class object.
Parent Constructor Constructor of a superclass called using super().
Partial Class A class split across files; Java does not support partial classes.
Partial Compilation Compiling only changed source files.
Partial Evaluation Optimizing code by evaluating known values early.
Parser Generator Tool that creates parsers from grammar definitions.
Parsing Tree Tree structure representing parsed source code.
Parse Tree Hierarchical representation produced during parsing.
Passivation Temporarily storing an object's state for later activation.
Path Interface NIO interface representing file system paths.
Path Variable Variable representing part of a resource path (common in frameworks).
Path Separator Character separating paths (: or ; depending on OS).
Path Element Individual part of a file path.
Path Normalization Simplifying paths by removing unnecessary components.
Path Resolution Converting a relative path into an absolute path.
Path Traversal Moving through directories using file paths.
Path Watcher Component that monitors file changes.
Pattern API Regular expression API in java.util.regex.
Pattern Object Compiled representation of a regular expression.
Pattern Matcher Object that applies a pattern to input text.
Pattern Flag Option that modifies regex behavior.
Pattern Group Part of a regex enclosed in parentheses.
Pattern Compilation Converting regex text into a usable Pattern object.
Permission Collection Collection of security permissions.
Permissions Class Class representing a set of permissions.
Policy Class Represents security policy information.
Policy Provider Implementation that supplies security policies.
Policy File File containing security permissions.
Privileged Action Operation executed with elevated security permissions.
PrivilegedActionException Exception thrown during privileged operations.
PrivilegedExceptionAction Interface for privileged actions that may throw exceptions.
Primitive Array Array storing primitive values like int[] or char[].
Primitive Conversion Conversion between primitive data types.
Primitive Narrowing Conversion Conversion from larger primitive type to smaller one.
Primitive Widening Conversion Automatic conversion from smaller primitive type to larger one.
Primitive Wrapper Object form of primitive types (Integer, Double, etc.).
Print Service API for managing printing operations.
PrintJob Represents a printing task.
PrinterJob Class used to control printer jobs.
Printer Graphics Graphics context used for printing.
PrintRequestAttribute Attribute describing a print request.
PrintService Interface representing a printer service.
PrintServiceLookup Utility for finding print services.
ProcessBuilder.Redirect Defines input/output redirection for processes.
ProcessBuilder.Redirect.Type Type of process redirection.
ProcessHandle Interface for controlling operating system processes.
Process ID (PID) Unique identifier of a running process.
Process Tree Parent-child relationship structure of processes.
Process Stream Input/output streams connected to a process.
Program Argument Value supplied to a program when it starts.
Program Execution Running a compiled Java application.
Program Structure Organization of classes, methods, and packages.
Programming Model Style or approach used to write programs.
Programming Paradigm General approach to programming (OOP, functional, etc.).
Projection Selecting specific data elements from a collection or database result.
Promise Asynchronous result concept (common in other languages/frameworks).
Property File File storing configuration values as key-value pairs.
Property Editor Tool for editing JavaBean properties.
Property Pattern Naming convention for JavaBean properties.
Protocol Handler Component that handles communication protocols.
Proxy Class Dynamically generated class that forwards method calls.
Proxy Interface Interface implemented by a dynamic proxy.
Proxy Object Object that controls access to another object.
Proxy Server Intermediate server handling network requests.
Proxy Authentication Authentication process when using a proxy.
Public API Publicly available methods/classes intended for use by developers.
Public Class Class accessible from any package.
Public Method Method accessible from anywhere.
Public Static Void Main Standard entry point method of a Java program.
Put Method Method used to add key-value pairs in maps.
PutIfAbsent Map method that adds a value only if the key is missing.
Push Operation Adding an item to a stack.
Pop Operation Removing an item from a stack.
PushbackReader Reader that allows characters to be pushed back into the stream.
PushbackInputStream Input stream allowing bytes to be pushed back.
Pseudo Code Informal algorithm description.
Pseudo Random Generator Algorithm that generates deterministic random-like values.
Pseudo Thread Simulated thread concept in some systems.
