# JavaPedia ☕

JavaPedia is a responsive Java Dictionary and Learning Portal built using HTML, CSS, and Bootstrap.

## Features

- 📚 About Java
- 📖 Java Dictionary (A–Z)
- 📂 Java Categories
- 📝 Detailed Java Concepts
- 🎨 Responsive Bootstrap Design
- 🔍 Organized Learning Resources

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons

## Pages

- Home
- About Java
- Categories
- Dictionary
- About Us
- A–Z Dictionary

## Project Highlights

- Responsive Design
- Sticky Navigation
- Bootstrap ScrollSpy
- Clean UI
- Educational Content
