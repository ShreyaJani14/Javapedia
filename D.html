D

Terms Defination
Data Abstraction The process of hiding implementation details and showing only essential features to the user.
Data Encapsulation The process of binding data and methods into a single unit called a class.
Data Hiding Restricting direct access to data using access modifiers.
Data Structure A way of organizing and storing data efficiently.
Data Model A representation of how data is organized and managed in an application.
Data Source An object used to obtain and manage database connections efficiently.
Data Transfer Object (DTO) An object used to transfer data between different layers of an application.
DataInput An interface used to read primitive data types from an input stream.
DataInputStream A stream used to read primitive Java data types from an input stream.
DataOutput An interface used to write primitive data types to an output stream.
DataOutputStream A stream used to write primitive Java data types to an output stream.
Database An organized collection of data that can be stored and accessed electronically.
Database Connection A link established between a Java application and a database.
DatagramPacket A class used to send and receive UDP network packets.
DatagramSocket A socket used for UDP communication between applications.
Date A class used to represent date and time values.
DateFormat A class used to format and parse dates.
DateTimeFormatter A class used to format and parse date-time objects in Java 8 and later.
DayOfWeek An enum representing the seven days of the week.
Deadlock A situation where two or more threads wait indefinitely for each other.
Debugging The process of finding and fixing errors in a program.
Debugger A tool used to test and debug programs by examining execution flow.
Declaration A statement used to define variables, methods, classes, or objects.
Decorator Pattern A design pattern used to add functionality to objects dynamically.
DecimalFormat A class used to format decimal numbers according to a specified pattern.
Decrement Operator The -- operator used to decrease a variable's value by one.
Default Access Modifier An access level that allows access only within the same package.
Default Constructor A constructor automatically provided by Java if none is defined.
Default Method A method in an interface that includes implementation using the default keyword.
Default Value The initial value automatically assigned to variables by Java.
DefaultListModel A model used to manage data in a JList component.
DefaultTableModel A model used to manage data in a JTable component.
DefaultTreeModel A model used to manage data in a JTree component.
DefaultMutableTreeNode A node used in tree structures for Swing applications.
DelayQueue A blocking queue where elements become available only after a specified delay.
Deque A double-ended queue that allows insertion and deletion at both ends.
Deque Interface An interface representing a double-ended queue.
DequeIterator An iterator used to traverse elements of a deque.
Dependency A relationship where one class relies on another class.
Dependency Injection A technique where required objects are provided externally rather than created internally.
Deprecated An annotation indicating that a class or method should no longer be used.
Descriptor An object that provides metadata information about another object.
Deserialization The process of converting a byte stream back into an object.
Design Pattern A reusable solution to commonly occurring software design problems.
Destroy() An applet lifecycle method called before an applet is removed from memory.
Dialog A GUI window used to interact with the user and display messages.
Diamond Operator (<>) A feature that simplifies generic type declarations.
Dictionary A legacy class used to store key-value pairs.
DigestInputStream A stream that updates a message digest while reading data.
DigestOutputStream A stream that updates a message digest while writing data.
Dimension A class representing width and height values.
Direct Buffer A buffer allocated outside the Java heap for faster I/O operations.
Disabled Component A GUI component that cannot receive user input.
Directory A location used to store files and folders.
Dispatch The process of selecting and invoking the correct method at runtime.
Display Mode A class representing screen resolution and color settings.
Distributed Application An application whose components run on multiple networked systems.
Division Operator The / operator used to perform division operations.
do-while Loop A loop that executes at least once before checking its condition.
Doclet A program used with Javadoc to generate custom documentation.
Document A representation of an XML or HTML document in memory.
Document Object An object representing a document within the DOM structure.
Document Object Model (DOM) A programming interface used to represent and manipulate XML or HTML documents.
DocumentBuilder A class used to obtain DOM parsers for XML processing.
DocumentBuilderFactory A factory class used to create DocumentBuilder instances.
DocumentListener An interface used to monitor changes in text documents.
DOM Parser A parser that loads an entire XML document into memory as a tree structure.
Double A primitive data type used to store double-precision floating-point numbers.
Double Buffering A technique used to reduce flickering in graphical applications.
DoubleBuffer A buffer used to store and manipulate double values.
DoubleStream A stream specialized for processing double values.
DoubleSummaryStatistics A class used to collect statistical information about double values.
Downcasting The process of converting a superclass reference into a subclass reference.
Download Manager A component that manages and controls file downloads.
Drag and Drop A GUI feature that allows users to move objects using the mouse.
DrawString() A Graphics method used to display text on a component.
Drawable Object An object that can be rendered graphically on the screen.
Driver A software component that enables communication between Java and a database.
DriverManager A JDBC class used to manage database drivers and connections.
DropTarget A class used in drag-and-drop operations.
DropTargetListener An interface used to receive drag-and-drop events.
DTD (Document Type Definition) Defines the structure and legal elements of an XML document.
Duration A class in the Java Time API representing a time-based amount of time.
DurationUnit A unit used for measuring durations in date-time operations.
Duplicate Key A repeated key value that is generally not allowed in maps.
Dynamic Array A resizable array structure such as ArrayList.
Dynamic Binding The process of determining the method to invoke at runtime.
Dynamic Dispatch A mechanism by which an overridden method is selected at runtime.
Dynamic Loading Loading classes into memory during program execution.
Dynamic Memory Allocation Allocating memory during runtime instead of compile time.
Dynamic Method Dispatch Runtime selection of overridden methods based on the actual object type.
Dynamic Programming An algorithmic technique that stores intermediate results to solve problems efficiently.
Dynamic Proxy A class generated at runtime that implements interfaces dynamically.
Daemon Thread A background thread that provides services to user threads.
default Specifies a default case in a switch statement or a default method in an interface.
do Used to create a do-while loop.
double A primitive data type used to store double-precision floating-point numbers.
